## 🎮 HOW TO USE

### To Start
1. Open terminal in `practice_session` folder
2. Run: `python app.py`
3. Open `index.html` in your browser
4. Click the chat button to start

### To Chat
1. Click the purple chat button (bottom-right)
2. Type a question
3. Press Enter to send
4. View the bot's response

### To Customize
1. Edit `app.py` to change personal info
2. Edit `index.html` to update content
3. Edit `style.css` to change colors
4. Add new intents in `app.py`
